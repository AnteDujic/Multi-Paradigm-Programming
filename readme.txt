Contents of this folder:

- Folders:
	- c - contains the application written in C (procedural paradigm)
	- python - contains the application written in Python (procedural paradigm)
	- python_OOP - contains the application written in Python (OOP)

- stock.csv - file which is read to create shop

- Customer files which are read by the applications
	- ante.csv - customer that doesn't have enough money for their shopping
	- dominic.csv - customer that wants more items then available on the shop stock (11 Big Bags vs 4 on stock)
	- john.csv - customer with no restrictions

- report.pdf - report on the applications design, the similarities and differences in the paradigm approaches

- readme.txt
